클라우드 입찰 솔루션에 대한 자주 묻는 질문

네, PWA(Progressive Web App) 기술을 적용하여 오프라인 모드를 지원합니다. 기본적인 분석 기능은 인터넷 연결 없이도 사용 가능하며, 연결이 복구되면 자동으로 동기화됩니다.

ISO 27001 인증을 받은 AWS 클라우드에서 운영되며, 256bit SSL 암호화와 정기적인 보안 감사를 통해 은행 수준의 보안을 제공합니다.

네, 실시간 협업 기능을 지원합니다. 여러 사용자가 동시에 접속하여 작업할 수 있으며, 권한 설정을 통해 접근 범위를 제어할 수 있습니다.

네, 반응형 웹 디자인으로 PC와 동일한 모든 기능을 스마트폰과 태블릿에서도 완벽하게 사용할 수 있습니다. 터치 최적화 인터페이스로 더욱 편리합니다.
